# Basic Spring Web Project
이 프젝은 스프링 기반 웹 애플리케이션 설정 및 기능을 작성 및 테스트 하기 위해 설계된 웹 애플리케이션임

## 특징
- build Tool인 Gradle 사용 및 환경 구성
- h2 DB의 로컬저장소 의존성 추가
- 스프링부트를 이용한 간편설정
- Vue앱의 정적빌드를 통한 프로젝트내 정적 빌드파일 생성
- 순수 자바코드를 통한 HTTP 서버 통신
- 카카오 페이 단일 결제 API 구현 + 간단한 화면
